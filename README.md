# 🎮 상용 게임 분석 보고서

본 문서는 상용 게임의 핵심 메커니즘과 플레이어 리텐션 전략을 분석한 기록입니다.

---

## 🔍 분석 대상 게임
* **게임 제목**: 몬스터 헌터 (Monster Hunter) 시리즈
* **장르**: 헌팅 액션 (Hunting Action)

---

## 🔄 코어 루프 (Core Loop)
몬스터 헌터의 핵심 반복 구조는 **'준비 - 수렵 - 강화'**의 선순환 구조로 이루어져 있습니다.

1.  **의뢰 수주**: 길드로부터 특정 몬스터의 토벌 의뢰를 받습니다.
2.  **몬스터 수렵**: 타겟 몬스터를 추적하고 고유의 액션으로 제압합니다.
3.  **장비 제작**: 완수 보상 및 몬스터 사체에서 갈무리한 소재로 더 강력한 장비를 만듭니다.
4.  **다음 단계**: 강화된 장비로 더 높은 난이도의 몬스터에게 도전합니다.

---

## ✨ 핵심 재미 요소 (Fun Factors)
플레이어를 반복하게 만들고 몰입시키는 3가지 핵심 요소입니다.

* **묵직한 타격감**: 무기마다 설정된 고유의 역경직과 연출을 통해 실제 거대 생명체를 타격하는 듯한 피드백을 제공합니다.
* **살아있는 생태계**: 몬스터 간의 영역 다툼, 실시간 환경 변화를 통해 단순한 보스전이 아닌 '세계관 속의 사냥'이라는 몰입감을 선사합니다.
* **다양한 액션 스펙트럼**: 14종 이상의 무기군이 각기 전혀 다른 조작 방식과 특징을 가지고 있어, 무기를 바꾸는 것만으로도 새로운 게임 경험을 제공합니다.

## ⚖️ 유사 게임 비교 분석
### 참조 게임: [그랑블루 판타지 리링크]

| 구분 | 몬스터 헌터 시리즈 | 그랑블루 판타지 리링크 |
| :--- | :--- | :--- |
| **공통점** | 최대 4인 협동 플레이, 거대 보스 토벌 위주의 진행, 확실한 타격감 피드백 | |
| **차이점 (액션)** | **무게감 중심**: 무기별 딜레이와 위치 선정 중요, 환경 요소 적극 활용 | **속도감 중심**: 저스트 회피/가드, 스킬 체인(연계기) 등 화려한 연출 |
| **차이점 (시스템)** | **생태계 상호작용**: 오픈 월드 지형 이용, 몬스터 간 세력 다툼 | **패턴 공략 중심**: 제한된 구역에서의 기믹 파훼, 보스 그로기 유발 |
| **차이점 (동료)** | **아이루 & 자유도**: 솔로 플레이 시 동반자 아이루와 함께하며 원하는 무기군 조합 가능 | **파티 빌딩**: 캐릭터 기반 스킬 세팅, 팀원 간 시너지 및 궁극기 연계 강조 |

---

> **분석 요약**: 몬스터 헌터가 '사냥꾼으로서의 시뮬레이션과 몰입'에 집중한다면, 리링크는 '캐릭터들의 협동과 화려한 보스 레이드'의 재미를 극대화한 게임입니다.
